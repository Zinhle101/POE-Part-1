# CyberSafe Chatbot

A cybersecurity educational chatbot built with C# that teaches users about password security, phishing, and suspicious links.

## Features

-  Interactive chatbot named "Chatty"
-  Password security tips
-  Link safety guidance
-  Phishing awareness education
-  Color-coded console output
-  Welcome audio greeting

## How to Use

1. Run the application
2. Enter your name when prompted
3. Ask Chatty about:
   - **Passwords** - Type "password" or "strong password"
   - **Links** - Type "link", "website", or "url"
   - **Phishing** - Type "phishing" or "email"
4. Type "exit" to quit

## Example Questions

- "How do I create a strong password?"
- "Tell me about phishing emails"
- "What about suspicious links?"

## Technologies Used

- C# (.NET 9.0)
- System.Media for audio playback
- Console application

## Installation

1. Clone or download this repository
2. Open in Visual Studio Code
3. Run: `dotnet build`
4. Run: `dotnet run`
